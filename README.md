# Overview

Parses and encodes Plist XML documents between a DOM and XML data.


# Usage

## Basic Usage

```js
import {Plist} from '@shockpkg/plist-dom';

const plist = new Plist();
plist.fromXml(plistXml);
console.log(plist);
console.log(plist.toXml());
```
